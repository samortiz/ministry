The Way of the Cross - The Way of Life

E631 C464 K464 S297 T631

Lyrics:Witness Lee
Music:Winfield S. Weeden
Meter:8.7.8.7. with chorus

Hymn using same tune:

#441

Verses:  1   2   3 

 
 1 

If I'd know Christ's risen power,

I must ever love the Cross;

Life from death alone arise;

There's no gain except by loss.
 

If no death, no life,

If no death, no life;

Life from death alone arises;

If no death, no life.
 
 2 

If I'd have Christ formed in me,

I must breathe my final breath,

Live within the Cross's shadow,

Put my soul-life e'er to death.
 

If no death, no life,

If no death, no life;

Life from death alone arises;

If no death, no life.
 
 3 

If God thru th' Eternal Spirit

Nail me ever with the Lord;

Only then as death is working

Will His life thru me be poured.
 

If no death, no life,

If no death, no life;

Life from death alone arises;

If no death, no life.
